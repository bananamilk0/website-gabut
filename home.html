Program Language

Di situs ini developer tentu menggunakan bahasa pemrograman yang sangat familiar karena mudah untuk difahami



HTML

HTML adalah bahasa yang udah dipelajari dan difahami. HTML adalah bahasa pertama dan yang terpenting yang akan ditemui oleh siapa pun yang mempelajari pengembangan web.

Bahasa ini hanya terdiri dari beberapa tag yang memiliki tujuan tertentu. Seseorang dapat dengan mudah memahami kode orang lain dan melakukan penyesuaian jika perlu.

Salah satu keberuntungan terbesar menggunakan HTML adlah gratis dan tidak memerlukan perangkat lunak khusus. HTML tidak memerlukan plugin apa pun, jadi seseorang tidak perlu berurusan denngan plugin saat bekerja dengan perangkat lunak apa pun.

HTML didukung oleh hampir semua browser di seluruh dunia seperti Google, Safari, Opera, dll. Jadi tidak perlu khawatir tentang situs web yang dibangun dalam HTML untuk dukungan browser karena situs web akan muncul di semua browser.

Dan masih banyak lagi kelebihan-kelebihan yang dimiliki HTML.

CSS

CSS memastikan identitas visual yang konsisten di seluruh situs web atau beberapa halaman web. Dengan menggunakan satu berkas CSS eksternal, gaya yang kohesif dapat di pertahankan, meminimalkan duplikasi

Perawatan lebih mudah dan pembaruan efisien, pemisahan konten dan gaya dalam CSS menyederhanakan pengelolaan halaman web dan penerapan pembaruan. Dengan memodifikasi beberapa baris kode dalam berkas CSS pusat, anda dapat dengan cepat menyesuaikan font, warna, dan tata letak tanpa perlu mengedit setiap elemen HTML. hal ini menghasilkan proses pemeeliharaan yang lebih efisien dan bebas repot

Hemat waktu dan pemuatan lebih cepat, dengan memanfaatkan file CSS eksternal, browser web dapat menyimpan informasi gaya dalam chace, sehinggan waktu pemuatan halaman menjadi lebih efisien dan cepat.

JAVA SCRIPT

Sejak pertama kali diluncurkan, banyak kelebihan yang ditawarkan oleh bahasa pemrograman javscript iniDalam mempelajari bahasa pemrograman,

terkadang orang awam membutuhkan waktu yang lama. Setiap bahasa pemrograman pasti memiliki tingkat kesulitan yang berbeda-beda. JavaScript adalah bahasa pemrograman yang termasuk mudah dipelajari sehingga menjadi pilihan para pengembang. Bahasa pemrograman ini memiliki syntax yang singkat dan sederhana dibandingkan bahasa pemrograman lainnya. Tanpa harus kuliah pun, Anda bisa mempelajari JavaScript melalui tutorial di internet atau juga membaca ebook.

JavaScript memiliki kelebihan yaitu bersifat multi platform, yang artinya bisa dijalankan di berbagai sistem operasi. Bahasa pemrograman ini bisa digunakan di sistem operasi Windows, Linus, Android, dan lainnya.

Bahkan, bahasa pemrograman ini bisa digunakan di perangkat yang tidak menggunakan sistem operasi tertentu. Contohnya yaitu handphone atau smartphone.
